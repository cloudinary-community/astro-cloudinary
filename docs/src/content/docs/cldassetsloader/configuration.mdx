#### `assetType`

The type of the assets you want to query.

**Examples**

<CodeBlock alwaysExpanded>
```jsx
assetType: 'video'
```
</CodeBlock>

[Learn more about Asset Types](https://cloudinary.com/documentation/upload_parameters#asset_types) on the Cloudinary docs.

#### `deliveryType`

The delivery type of the assets you want to query.

**Examples**

<CodeBlock alwaysExpanded>
```jsx
deliveryType: 'private'
```
</CodeBlock>

[Learn more about Delivery Types](https://cloudinary.com/documentation/upload_parameters#delivery_types) on the Cloudinary docs.

#### `folder`

The folder the assets should be queried from.

**Examples**

<CodeBlock alwaysExpanded>
```jsx
folder: 'samples/landscapes'
```
</CodeBlock>

#### `limit`

The maximum amount of results to query.

**Examples**

<CodeBlock alwaysExpanded>
```jsx
limit: 20
```
</CodeBlock>

#### `type`

The type of query to make for assets, including list or search.

This primarily differs in how the requests are made, using the Admin API or the Search API.

**Examples**

<CodeBlock alwaysExpanded>
```jsx
type: 'search'
```
</CodeBlock>
